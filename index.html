Interests

 I work in the field of synthetic aperture radar(SAR)/Polarimetric SAR data processing, target detection and recognition, remote sensing image processing and deep learning theory. Currently, I focus on the following research topics:
  • SAR/PolSAR image scattering feature analysis
  • Fast superpixel segmentation and merging for SAR/PolSAR data
  • Target detection and recognition from SAR/PolSAR data
  • Image matching for SAR images
★长期从事合成孔径雷达图像解译算法研究，包括SAR和全极化SAR的散射特征提取、极化目标分解、图像分割和分类、目标检测与鉴别、变化检测分析等应用。★
